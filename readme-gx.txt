how to use:
place guxt folder in "dist" folder
folder must be called guxt

game should be modded to disable pximg scrambling and use bmps instead

hotkeys:


J & K: switch stage
Arrowkeys: Scroll
Home & End: Scroll
Pgdn, PgUp: Scroll
minus & Equals: Zoom stage
[ and ]: zoom tileset
' and : zoom entities (broken)
CTRL-S: Save

i: place entities

CTRL-Z CTRL-Y

F11: fullscreen
